Manage events using Eventbrite with WordPress (WIP)

== Description ==

Eventbrite for WordPress is a complete solution for events management using their API.
You can push and pull events from Eventbrite directly from WordPress. Uses post types.

Plugin sponsored by [Vivanista.com](http://vivanista.com)
